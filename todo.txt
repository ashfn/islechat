Channel commands

global


new|make|create <name> -> name is optional but makes a new channel with you as the owner
public -> makes the current channel public if you are the owner of it (anyone can join)
private -> makes the current channel private if you are the owner of it (invite only)
invite <user> -> invites the user to the channel if it is private (shows error if public)
uninvite <user> -> revokes a users invite to the channel
join <channel> -> joins the channel if it is public or you are invited
kick <user> -> kicks the user from the channel if your the owner
ban <user> -> bans he user from the channel (only works if the channel is public)
delete -> deletes the channel if your the owner
banner <text> changes the banner of the channel
readonly toggles whether the channel is read only
